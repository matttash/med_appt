# StayHealthy Medical Appointment Booking Website

## Project Description
StayHealthy is a full-stack medical appointment booking web application that allows patients to connect with doctors, book appointments, and manage healthcare interactions online.

## Features
- User registration and login
- Doctor search and profiles
- Appointment booking system
- Patient reviews
- Notifications system
- Profile management

## Technologies Used
- React (Vite)
- Node.js
- Express.js
- MongoDB
- HTML
- CSS
- JavaScript

## Installation Instructions

### 1. Clone the repository
git clone https://github.com/matttash/med_appt.git

### 2. Install frontend dependencies
npm install

### 3. Start frontend
npm run dev

### 4. Start backend (in another terminal)
cd server
npm install
npm run dev
